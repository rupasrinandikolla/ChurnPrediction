# Customer Churn Prediction

This project predicts whether a telecom customer is likely to leave the company using Machine Learning.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Flask
* HTML
* CSS
* GitHub

## Features

* Customer churn prediction
* Machine learning model training
* Web interface using Flask
* Prediction results displayed on webpage

## Project Structure

* train.py - Trains the machine learning model
* app.py - Runs the Flask application
* dataset/Churn.csv - Dataset used for training
* model/churn_model.pkl - Saved machine learning model
* templates/index.html - Web page

## How to Run

1. Run train.py
2. Run app.py
3. Open http://127.0.0.1:5000 in browser
